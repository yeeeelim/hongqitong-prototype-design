# 红骑通原型设计

这是红骑通外卖骑手服务平台的高保真原型设计文件。

## 项目说明

红骑通是一个专注于为外卖骑手提供全方位的权益保障和职业发展支持的移动应用平台。包含以下核心功能模块：

- 政策与法律支持
- 保险服务
- 成长学院
- 积分激励
- 速通服务
- 活力社区

## 技术栈

- HTML5
- Tailwind CSS
- Font Awesome 图标
- JavaScript

## 设计规范

- 遵循 Apple Human Interface Guidelines
- 主色调：品牌红色 (#E53935)
- 辅助色：蓝色 (#1976D2)
- 支持暗黑模式
